Installation
------------
Install [Make], [Docker] and [docker-compose] to be able to start a project

Starting
------------
To start a project
```
$ Make run
```

To reload a project
```
$ Make reload
```
To stop a project, and delete all containers, images, and volumes
```
$ Make stop
```
To reload a project
```
$ Make reload
```
